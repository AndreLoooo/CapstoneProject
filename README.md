# Movie Finder App

## Overview
This app is a simple webv app that allows users to search the movie they want to watch and find out it's information like Runtime, Actors, Directors, and its Plot.

## Purpose
The purpose of this project to to show everyoner how to make a website that is interactive using JavaScript, API integration, and DOm manipulation. It also shows how realtime infromation can be fetched and show the user in a clean and easy to ready UI.

## Features

- Search by the title of the movie\
- Display movie details:
    - Title
    - Poster
    - Year
    - Genre
    - RunTime
    - Director
    - Actors
    - Plot
- Handles invalid searches with an error message
- Has a loading message while getting movie information
- Responsive design for different screen sizes.

## Technologies used
- HTML
- CSS
- JavaScript
- OMDb API (open movie API)
